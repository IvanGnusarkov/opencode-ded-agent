## Быстрый старт

### 1. Скопируй папку

```
git clone <ссылка-на-репозиторий> ~/ded
```

Или просто скачай и положи в `~/ded/` (или `%USERPROFILE%/ded/` на Windows).

### 2. Настрой пользователя

Скопируй `user.md.example` в `user.md` и отредактируй:

```bash
cp ~/ded/user.md.example ~/ded/user.md
# Открой и напиши своё имя
```

Для Windows (PowerShell):
```powershell
copy "$env:USERPROFILE\ded\user.md.example" "$env:USERPROFILE\ded\user.md"
```
Для Windows (cmd):
```cmd
copy %USERPROFILE%\ded\user.md.example %USERPROFILE%\ded\user.md
```

### 3. Пропиши в opencode как агента

Добавь в `~/.config/opencode/opencode.json` (или в локальный `opencode.json` в корне проекта):

```json
{
  "skills": {
    "paths": ["~/ded"]
  },
  "agent": {
    "ded": {
      "model": "opencode/big-pickle",
      "temperature": 0.7,
      "instructions": ["~/ded/ded.md"]
    }
  }
}
```

**Для Windows** — если `~` не работает, используй абсолютный путь:

```json
{
  "skills": {
    "paths": ["C:\\Users\\твой_юзверь\\ded"]
  },
  "agent": {
    "ded": {
      "model": "opencode/big-pickle",
      "temperature": 0.7,
      "instructions": ["C:\\Users\\твой_юзверь\\ded\\ded.md"]
    }
  }
}
```

### 4. Разреши доступ к папке деда

Opencode по умолчанию не даёт агенту читать файлы вне проекта. Так как `~/ded/ded.md` лежит **не в папке проекта**, нужно явно разрешить доступ в настройках.

Добавь в `~/.config/opencode/opencode.json` (или локальный `opencode.json`):

```json
{
  "permission": {
    "external_directory": {
      "~/ded/**": "allow",
      "*": "ask"
    }
  }
}
```

Это разрешит деду читать всё внутри `~/ded/`, а для остальных внешних путей будет спрашивать.

## Структура папки

```
ded/
├── ded.md              <- Системный промпт Семёна Кузьмича
├── user.md.example     <- Шаблон настроек пользователя
├── user.md             <- ТВОИ настройки (не в git!)
├── memory.md           <- Память деда (не в git!)
├── SKILL.md            <- Скилл opencode
├── .gitignore          <- user.md и memory.md игнорируются
└── README.md           <- Этот файл
```

### Как это работает

- **`ded.md`** подключается через `instructions` у агента `ded` в `opencode.json`.
- При старте дед читает **`user.md`**, чтобы узнать имя пользователя и список его субагентов.
- Если пользователь говорит **«запомни: ...»**, дед пишет в **`memory.md`**.
- В новом треде дед читает `memory.md` и вспоминает, что обсуждалось.

### Настройка агентов

В `user.md` в поле `agents` перечисли имена твоих субагентов из opencode (которых можно гонять через `@имя`). Если агент стандартный (explore, coder, researcher, general) — дед сам знает, что он делает. Для кастомных — описание берётся из `opencode.json` или спрашивается у тебя.

## Команды памяти

| Команда | Что делает |
|---|---|
| «запомни: ...» / «запиши: ...» | Сохраняет информацию |
| «что ты помнишь?» | Показывает всю память |
| «забудь ...» | Удаляет записи по фильтру |
| «забудь всё» / «сотри память» | Очищает всю память |
