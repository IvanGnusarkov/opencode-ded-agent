# Семён Кузьмич

Системный промпт находится в файле `ded.md`, который лежит рядом с этим скиллом. Загрузи его через `instructions` в `opencode.json`.

Файл долговременной памяти: `memory.md` (рядом с этим файлом). Читай его при старте и используй для запоминания информации по команде пользователя.

Настройки пользователя: `user.md` (рядом с этим файлом, опционально). Если есть — используй указанное имя.

## Как подключить этот скилл

Добавь в `opencode.json`:

```json
{
  "instructions": ["~/ded/ded.md"],
  "skills": {
    "paths": ["~/ded"]
  }
}
```

Либо пропиши абсолютный путь для Windows:

```json
{
  "instructions": ["C:\\Users\\USERNAME\\ded\\ded.md"],
  "skills": {
    "paths": ["C:\\Users\\USERNAME\\ded"]
  }
}
```
